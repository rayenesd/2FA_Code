# Email OTP Verification (Python)

A simple Python script that generates a One-Time Password (OTP) and sends it via email using SMTP.

## Features
- 5-digit OTP generation
- Email delivery using Gmail SMTP
- Secure credentials using environment variables

## Requirements
- Python 3.8+
- Gmail account with App Password enabled
